
# Weather App

An application which reads your current location and tells the exact Weather by getting data from weather API.

## API Reference

#### Get all items

```http
 https://api.openweathermap.org/data/2.5/weather?lat=${latitude}&lon=${longitude}&appid=${key2}
```
